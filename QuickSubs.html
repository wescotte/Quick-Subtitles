Quick Subtitles takes a data entry philosophy that you should minimize the frequency in which you take your hands off the keyboard. In order to achieve this the majority functionality has been mapped to keyboard. Here is a list of keys to control the program and their behavior.

Key

Behavior
F1

Bring up this help screen
ESC

Toggle video play state: Pause the video if playing and play the video if paused.
CTRL + I
CTRL + [

Set IN POINT to the current frame of the video.

You can also set the IN POINT when you type any text into the subtitle entry box.

CTRL + O
CTRL + ]

Set OUT POINT to the current frame of the video.

You can also set the OUT POINT when you hit enter in the subtitle entry box
CTRL + ,
CTRL + <

Clears the current IN POINT.
CTRL + .
CTRL + >

Clears the current OUT POINT.
ENTER

When typing up a subtitle hitting enter will add your subtitle to the project.
However, if no OUT POINT is assigned it will set set one based on the current frame of the video instead.

SHIFT + ENTER

Adds a line break to your subtitle
+ ←

Move the playhead to the previous IN POINT.

+ →

Move the playhead to the next IN POINT.

+ Page
Up

Undo last action.
Alt keyboard shortcut is F2
+ Page
Down

Redo last action.
Alt keyboard shortcut is F3
+ ↓

Adjusts the IN POINT of the nearest subtitle based on the current frame of the video. There are two cases on how this works

CASE 1: If the current video frame is between an IN and OUT POINT then the IN POINT will be adjusted to the later position

CASE 2: If the current video frame is between an OUT and IN POINT then the IN POINT will be adjusted to the earlier position
+ ↑

Adjusts the OUT POINT of the nearest subtitle based on the current frame of the video. There are two cases on how this works

CASE 1: If the current video frame is between an IN and OUT POINT then the OUT POINT will be adjusted to the earlier position

CASE 2: If the current video frame is between an OUT and IN POINT then the OUT POINT will be adjusted to the later position
ALT + ↓

Same behavior as + ↓ but when two consecutive subtitles share a common timecode (the first subtitle's OUT point is the same next subtitles IN point) only the IN point is adjusted leaving the first subtitles OUT point in the original position.

ALT + ↑

Same behavior as + ↑ but when two consecutive subtitles share a common timecode (the first subtitle's OUT point is the same next subtitles IN point) only the OUT point is adjusted leaving the second subtitles IN point in the original position.

CTRL + -

Slow down video play rate by 10%.
Alt keyboard shortcut is just - on the numeric keypad
CTRL + +

Speed up video play rate by 10%.
Alt keyboard shortcut is just + on the numeric keypad
TAB

"Rewind" to nearest IN POINT. If you tap it in rapid succession you can jump backwards more than one IN POINT. This feature is useful when entering subtitles while the video plays and you need to repeat the dialog.

You can manipulate multiple subtitles at once by using the checkboxes next to them. If you click any checkbox and then hold down theShift key and click another checkbox it will selet or deselect that range of checkboxes. You can also toggle selecting all the boxes by clicking the top most box in the header.

Adjust the IN and OUT points of a subtitle in 0.025 second increments up to 1.5 seconds in either direction.
Hold down SHIFT to adjust only the IN point and ALT to adjust only the OUT point.

Splits the subtitle into two subtitles each half the length of the original

Move the move cursor over the waveform and you can drag to scrub through your video.
Holding down SHIFT while dragging allows you to scrub faster.
